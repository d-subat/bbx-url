APIShort URL Microservice

To use the API send an URL to https://bbx-url.glitch.me/api/new/url, you will get a response in json format, containing the short URL.

Example 

      https://bbx-url.glitch.me/api/new/http://www.example.com
      
      {
        original_url:	"http://www.example.com",
        short_url:	"NDk0NzU9XzQ3NTkx"
      }
